Projects

Wuggy: The Productivity Pet

Wuggy: The Productivity Pet is a browser extension built for DS6 of Human-Computer Interaction at Williams College. The extension is built using HTML, CSS, and JavaScript. Within the extension, users are able to create to-do lists, complete tasks in their to-do lists, and "feed" their Wuggy submitted Canvas Assignments to progress the age of their virtual pet. The design document link for the project can be found here.

Book Catalog Website

Our book catalog website is a simple HTML website built for DS4 of Human-Computer Interaction at Wolliams College. The website is built using HTML, CSS, and JavaScript and uses the Bootstrap library. Within the website, users have some books loaded with information about authors and summaries, users can search and filter by genre. The design document link for the project can be found here.

2048 Python

Using Python, a group in my Williams College Human-Computer Interaction class developed a program that allows a user to play a rendition of 2048 with hand gestures. The program uses Python and the Mediapipe, cv2, and pyautogui libraries. The design document link for the project can be found here.
